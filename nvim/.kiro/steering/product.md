# Product Overview

This is a custom Neovim configuration built for development work across multiple languages and technologies. The setup focuses on providing a modern IDE-like experience with LSP support, fuzzy finding, file navigation, and git integration.

## Key Features

- LSP integration with Mason for automatic language server installation
- Fuzzy finding with Telescope for files, buffers, and live grep
- File tree navigation with Neo-tree
- Git integration (status, commits, branches)
- Auto-completion and formatting
- Support for multiple languages: Python, Rust, Lua, JavaScript/TypeScript, HTML, CSS, Docker, Terraform, YAML, Bash, GraphQL, SQL, and LaTeX
- Catppuccin color scheme
- Tmux integration for seamless navigation

## Target Use Cases

Infrastructure as Code (Ansible, Terraform, Kubernetes, Pulumi) and general software development with emphasis on DevOps tooling.
